# Terminal Realms: Dungeons

A retro-inspired multiplayer dungeon crawler based on MajorMUD.

## Overview

Terminal Realms: Dungeons brings the classic MajorMUD experience to modern platforms. Explore dungeons, battle monsters, collect loot, and level up your character in this text-based multiplayer adventure.

## Features (Planned)

- Classic text-based dungeon exploration
- Character classes and progression
- Real-time multiplayer interaction
- Monster battles and loot system
- Persistent world and character saves
- Modern web interface with retro aesthetics

## Technology Stack

- Backend: Node.js/Python
- Frontend: React with terminal-style UI
- Database: PostgreSQL
- Real-time: WebSockets
